== Description ==

WhereAreYouNow.com or WAYN.com is a great website that allows you to track your travels and meet like minded people. The WAYN Countries Visited Widget was created so that travelers with wordpress blogs can display their visited countries right in their sidebar.

== Installation ==

1. Upload post-clean folder to the `/wp-content/plugins/` directory OR through wp-admin,
upload post-clean.zip in plugin 'Add New' page.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Settings can be found under 'Setting' menu at the left side of wp-admin.

== Frequently Asked Questions ==

= I'm a member of WAYN but my profile is private. Can it be shown? =

No. Set your settings to public.

= I have visisted many countries, the flags are taking too much space, how can I make it smaller? =

You can set your flag icon to different sizes. Currently, we support 40px, 60px and
90px as default.

= I want to show more information from my profile. =

Future version will allow displaying more than the visit countries. Stay tuned!
